# 國王密使 IV：羅塞拉的冒險 — 中文說明書要點

> 整理自當年智冠科技／軟體世界中文說明書（掃描原件見發行方；本檔為重點索引與故事轉寫）。
> 譯名以此說明書為權威定名，全遊戲中文化一致沿用。

## 一、有關國王密使的傳說

在很久以前，有一個達文奇（Daventry）王國，國王叫做愛德華（Edward）。達文奇是個古老的王國，
歷經數千年、好壞國王輪替。愛德華是位非常好的國王，但年事已高又膝下無子。自從失去三大寶物後，
疾病與騷動逐漸侵襲這塊樂土。愛德華擔心自己死後情勢更糟，又苦無王位繼承人，於是派出他最鍾愛的
武士——卡拉漢爵士（Sir Graham）。

國王對卡拉漢說：你是我最勇敢、最值得信賴的武士，機智又勇武。我選你繼承王位，但你必須證明自己
有能力為王：找回多年前被巫術魔法竊走、藏於城堡之外的達文奇三大寶物。若成功，我死後王位便歸你；
若失敗，美麗的國土將落入黑暗勢力之手。

「卡拉漢爵士，祝你光榮凱旋而歸。」

卡拉漢歷經常人難以嘗試的艱險，帶著珍貴的三大寶物光榮歸來——這便是國王密使第一集《皇冠羅曼史》
（Quest for the Crown）所記載的故事。

## 二、卡拉漢的家庭

卡拉漢國王與美麗的少女法師蘭妮絲（Valanice，來自科利馬 Kolyma 王國）結為夫婦。（蘭妮絲曾被嫉妒的
巫婆海格達 Hagatha 擄走囚禁，卡拉漢前往科利馬、以三把魔法鑰匙救出她——此為國王密使第二集。）

兩年後，年輕的王后生下一對雙胞胎，一男一女：亞歷山大（Alexander）與羅塞拉（Rosella）。整個家庭
過著快樂平靜的日子……但好景不常。

一隻可怕的三頭惡龍朝達文奇襲來，年輕的羅塞拉公主被惡龍綁架。同時，遙遠之地住著一個惡毒的巫師
馬那那（Manannan）……解救羅塞拉、巫師的滅亡及王室重新團圓的過程，記載於國王密使第三集
《魔法生死鬥》。

## 三、本集故事簡介

失落多年的亞歷山大王子與羅塞拉公主平安歸來後，年老的卡拉漢國王決定將冒險家之帽傳給年輕一代。
就在他把帽子拋向孩子時，國王突感胸口一陣椎心之痛，倒地不起——「救救我！」

哀傷的羅塞拉跑出房間，在大廳對著牆上的魔鏡哭泣。這時魔鏡中浮現一個影像，一位仙女說道：

> 「我是仙女吉妮絲（Genesta），住在遙遠的塔米亞（Tamir）。那裡有一種神奇的樹，
> 一百年才結一顆果實，凡人吃了能恢復健康、強壯身體。」

吉妮絲告訴羅塞拉：她的法力正在消退——邪惡仙女樂樂蒂（Lolotte）趁她夜裡漫步時，偷走了她神奇的
**護身符（talisman）**。若 24 小時內找不回護身符，吉妮絲將會死去，而樂樂蒂會用護身符的力量為
塔米亞帶來更多災禍。

羅塞拉答應相助。吉妮絲以法力將她帶到塔米亞海邊，叮囑她：你能做到的比你想像的多，你一定能進入
樂樂蒂的領域……而那株神奇的樹，就在山那頭、一座四周沼澤環繞的小島上。

於是，羅塞拉的冒險就此展開——一日之內，取回護身符救吉妮絲、找到魔法果實救父王。

## 四、操作要點

- **打指令**：以「動詞＋名詞」輸入，如 `LOOK`（看）、`TAKE ROSE`（拿玫瑰）、`OPEN DOOR`（開門）。
- **選單**：畫面頂端點一下或按 ESC 叫出選單。
- **移動**：方向鍵操控羅塞拉行走。
- **存讀檔**：`SAVE GAME` / `RESTORE GAME`，遇危險或重要進度前務必存檔。

## 五、新手冒險守則（節錄）

1. **多觀察**：每到新場景先 `LOOK AROUND`，看到什麼就 examine，往往藏著線索。
2. **勤存檔**：這是動作冒險，會死。危險前後都存。
3. **畫地圖**：記下去過的地點、物件、危險區。塔米亞的人與生物會移動，同一地方下次未必相同。
4. **善用動詞**：遊戲聽得懂大量動詞（Take/Open/Look/Talk/Give/Use/Climb/Swim…）。
5. **山窮水盡別慌**：什麼都試試看，連「禁忌」的動作也試——玩遊戲不會被抓去關。
