# 國王密使IV：羅塞拉的冒險 EGA 繁中化 — 翻譯 subagent 統一指令

你在翻譯 Sierra 1988 經典動作冒險遊戲《King's Quest IV: The Perils of Rosella》(國王密使IV：羅塞拉的冒險) SCI0 EGA 原版。西方童話奇幻設定：公主羅塞拉一日之內在塔米亞奇幻大陸冒險，要取回護身符救仙女吉妮絲、找魔法果實救父王。**先讀完本檔再翻你那批**。譯名一律照下方權威詞表(取自當年智冠/軟體世界中文說明書)。

## 輸入 / 輸出格式

- 輸入是一個 `.tsv`：每行 `英文原文 <TAB> 英文原文`(col2 是待翻底稿)。
- 輸出同名 `.done` 檔：每行 `英文原文 <TAB> 繁體中文譯文`。
- **col1(英文原文)一字不改**——它是引擎查表 key，改了就對不上、退回英文。逐行對齊，行數不變。

## 硬規則(違反就壞)

1. **佔位符 `%s` `%d` `%c` `%x` 原位、原數保留**。中文語序可調，數量與型別不可增刪。例：`You have %d gold.` → `你有 %d 枚金幣。`(一個 %d)。
2. **控制碼保留**：`\r\n`/`\n`(硬換行)、開頭的 ` #` 選單標記、句中特殊序列，位置數量不動。
3. **繁體中文 + Big5 可編字**。不要簡體字、不要 Big5 打不出的罕字/顏文字。標點用中文全形：，。、：；！？「」『』…—。
4. **長度別暴增**(顯示有限)：譯文控制在原文語意內，別加註解、別擴寫。斷行交給引擎，**不要自己加 `\n`**(除非原文本來就有)。
5. 純功能性/系統字串照直譯(選單、錯誤提示、parser 回應)；有戲的對白/敘述才發揮語感。
6. **選單列帶前後空白**(如 `   Restore   `)：空白逐字保留，只譯可見字。

## 風格尺度(童話奇幻・自然口語)

- **總基調**：西方童話奇幻。敘述/旁白流暢書面體、帶點古典童話質感但不拗口；對白活潑自然、口語但不油。
- 主角羅塞拉：善良、勇敢、少女感。第二人稱一律「你」。
- 反派樂樂蒂：陰狠刻薄。仙女吉妮絲：溫柔慈祥。各角色語氣分明。
- **自然口語**：台式自然語感(像「怎麼會這樣」「別急」)，但**保留西方奇幻異國感**，別塞台灣時事梗/流行語。
- parser 回應(打錯指令)：簡潔、略帶遊戲旁白的俏皮，如 `That is not understood.` → 「這我看不懂。」
